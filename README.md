# PROCOM — Site Web Agence de Communication 360°

Site vitrine de PROCOM, agence conseil en communication 360° basée à Cotonou, Bénin.

## Stack technique

- **Framework** : Next.js (App Router)
- **Langage** : TypeScript / React
- **Thèmes** : Light / Dark mode avec persistance (localStorage)
- **Fonts** : TT Commons Pro
- **Déploiement** : Vercel

## Sections

- Hero avec animations
- Barre partenaires (marquee)
- À Propos
- Services (9 expertises)
- Process (4 étapes)
- Bénin Bouge (Nation Branding)
- Portfolio (filtrable par catégorie)
- FAQ (accordion)
- Newsletter
- Contact (formulaire + Google Maps)

## Fonctionnalités

- Toggle Light / Dark mode
- Scroll reveal animations
- Compteurs animés
- Portfolio filtrable
- FAQ accordion
- Menu mobile (clip-path animation)
- Bouton WhatsApp sticky
- Back-to-top
- SEO (Open Graph, Twitter Cards)

## Lancer en local

```bash
npm install
npm run dev
```
